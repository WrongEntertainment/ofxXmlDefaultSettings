ofxXmlDefaultSettings

General Information
ofxXmlDefaultSettings is an extension for openFrameworks to create default xml file. Load, set and save core settings.

openFrameworks core variables:

  • window position
  • window size
  • window title
  • framerate
  • fullscreen
  • cursor
  • escape quits app
  • log to file

The same xml stuff works for the following addons:
ofxOsc (ofxOscSender, ofxOscReceiver)

Also you can use the xml to store other variables.
