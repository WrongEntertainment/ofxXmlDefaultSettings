dependenciesofxXmlSettings


This openFrameworks addon create a default xml file to store openFrameworks settings like of ofSetFrameRate, ofSetWindowShape etc. The addon check, if a xml file exist to decide whether a file should be created or imported. If no file is found, the addon generate a default file and loads this one. You can found more information about the default parameter at the ofxXmlDefaultSettings constructor and the XML file.

XML file

A sample of the generated XML file. 

<ofxXmlDefaultSettings version="0.2.1" url="https://github.com/wrongentertainment/ofxxmldefaultsettings">
  <ofCore>
    <frameRate>60</frameRate>
    <fullscreen>0</fullscreen>
    <window x="0" y="44" width="1024" height="768">openFrameworks Application</window>
    <cursor>1</cursor>
    <escapeQuitsApp>1</escapeQuitsApp>
    <logToFile filepath="NULL" filename="logs.txt">1</logToFile>
  </ofCore>
</ofxXmlDefaultSettings>
