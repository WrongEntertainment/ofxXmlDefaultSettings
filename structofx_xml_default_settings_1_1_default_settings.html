Detailed Description

The DefaultSettings struct is used to store the setting parameter.
The attributes will be set by the ofxXmlDefaultSettings constructor.

Parameters

windowHeightThe default window height as an integer.

windowYThe default window y-position as an integer.
windowTitleThe default window title as a string.
frameRateThe default framerate of the application as an integer.
fullscreenThe default fullscreen settings as a boolean.
cursorThe default cursor settings as a boolean.
escapeQuitsAppThe default settings if escape quits the application. Set as a boolean.
logThe default log settings as a boolean.
logFilepathThe path where the log txt files should be saved.
