Member Enumeration Documentation

enum ofxXmlDefaultSettings::XmlSyntax

The enum is used to select a syntax string. All syntax strings will be stored at a string array. To change or get a syntax object, you can use the changeSyntax and getSyntax methods.
The ROOT, ROOT_VERSION and ROOT_URL syntax looks like

<ROOT version"ROOT_VERSION" url"ROOT_URL">

The CORE syntax looks like:

<CORE>
(...)
</CORE>

bool ofxXmlDefaultSettings::fileExist

True if a default xml file exist at the application launch. If a default xml file was generated, the fileExist parameter is also false. This can be used for advanced thinks like adding more tags to the xml file.
